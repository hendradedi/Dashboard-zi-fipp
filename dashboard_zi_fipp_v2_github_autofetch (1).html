📖 Cara Mendapatkan URL Raw GitHub

1
Upload file Excel ke GitHub

Buka repo GitHub → klik Add file → Upload files → pilih file .xlsx → Commit

2
Buka file di GitHub

Klik nama file .xlsx di daftar file repository

3
Klik tombol "Raw"

Di halaman file, klik tombol Raw di kanan atas. URL yang muncul di browser itulah yang dipakai.

4
Salin URL tersebut

Contoh hasil: https://raw.githubusercontent.com/hendradedi/zi-fipp/main/LKE-ZI-FIPP-2026.xlsx

5
Tempel di kotak di atas → Klik Ambil Data

Dashboard akan otomatis membaca file Excel dan menampilkan data. Setiap kali update Excel di GitHub, cukup klik 🔄 Reload.
